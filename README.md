# Notepad Plus
Notepad Plus is a feature-rich, web-based text editor with a modern interface, built using HTML, CSS, and JavaScript. It offers a sleek, customizable, and responsive notepad application with support for multiple tabs, rich text formatting, search and replace functionality, and theme toggling.

# Features
## Multi-Tab Support:
Create, manage, and switch between multiple tabs for editing different documents.

## Rich Text Formatting:
Apply bold, italic, and color formatting to text with keyboard shortcuts (Ctrl+B, Ctrl+I).

## Search and Replace:
Search for text with regex support, replace individual matches, or replace all occurrences.

## Theme Toggle:
Switch between light and dark themes, with the preference saved in local storage.

## Auto-Save:
Automatically saves tab content to local storage every 5 seconds to prevent data loss.

## Responsive Design:
Adapts to various screen sizes, including mobile devices, with a clean and intuitive layout.

## Particle Animation:
Subtle background particle effects for an engaging user experience.

## Glitch Title Animation:
A dynamic title animation with glitch effects for visual appeal.

## Undo/Redo:
Supports undo and redo operations for text editing.

## File Saving:
Save content as plain text (.txt) or formatted HTML (.html) files.

## Clear Cache:
Option to clear all locally stored tabs and reset the application.

## Zoom Support:
Adjust editor font size using Ctrl + mouse wheel.

## Clipboard History:
Stores up to 10 recent clipboard entries for easy access.

## Statistics:
Displays word, character, and line counts for the active tab.

# Getting Started
## Prerequisites
A modern web browser (Chrome, Firefox, Safari, Edge, etc.)
No additional dependencies or server setup required, as the application runs entirely in the browser.

## Installation
## Clone the Repository:
git clone https://github.com/mahmed-eng/notepad-prime.git

## Open the Application:
Navigate to the project directory.
Open index.html in a web browser to start using Notepad Plus.

## Usage
### Creating a New Tab:
Click the "New Tab" button to create a new document tab.
Each tab is auto-saved to local storage.

## Editing Text:
Type or paste text into the editor.
Use the toolbar buttons or shortcuts (Ctrl+B for bold, Ctrl+I for italic) to format text.
Select a color from the color picker to change text color.

## Search and Replace:
Enter a search term in the "Search" input field.
Use the "Find" button to locate matches or enable regex for advanced searches.
Use "Replace" or "Replace All" to modify text.

## Saving Files:
Click the "Save" button to open the save modal.
Enter a file name and choose a format (.txt or .html) to download the file.

## Theme Switching:
Click the "Theme" button to toggle between light and dark modes.

## Clearing Content:
Use the "Clear" button to clear the active tab's content (with confirmation).
Use the "Clear Cache" button to remove all locally stored tabs (with confirmation).

## File Structure:
index.html: The main HTML file containing the structure, styles, and JavaScript logic for the application.
No external dependencies or additional files are required.
